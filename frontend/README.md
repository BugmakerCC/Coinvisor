# AI 对话平台

基于 [Ant Design X](https://ant-design-x.antgroup.com/index-cn) 开发

## 技术栈

- React
- Ant Design X
- Ant Design
- TypeScript
- UmiJS

## 环境要求

- Node.js >= 14.0.0
- npm >= 6.14.0
- 现代浏览器 (不支持 IE)

## 开发

### 安装依赖

```bash
npm install
```

或者

```bash
yarn
```

### 启动项目

```bash
# 本地开发，默认开发环境
npm start

```

### 项目构建

```bash
# 构建
npm run build

```

## 项目结构

```bash
src/
├── pages/ # 页面组件
├── services/ # API 服务
├── utils/ # 工具函数
└── ...
```
